# Thesis Capstone (Firebase + Vue)

This project is now a **Vite + Vue 3 SPA** intended to be hosted on **Firebase Hosting** and use **Firebase Auth + Realtime Database** as the backend.

## Local dev

1. Install dependencies:
   - `npm install`
2. Run dev server:
   - `npm run dev`

## Build

- `npm run build`
- Output goes to `dist/`

## Firebase deploy (Hosting)

1. Set your Firebase project id in `.firebaserc`
2. Create a `.env` (see `.env.example`) with your Firebase web config values
3. Deploy:
   - `firebase deploy`
